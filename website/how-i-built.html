How I Built This Page

A serverless resume website with fully automated CI/CD deployment on AWS

Architecture Overview

This resume website is hosted entirely on AWS using a serverless architecture. Every push to the main branch automatically deploys to both staging and production environments.

Tech Stack

AWS CDK

Infrastructure as Code using TypeScript. Defines all AWS resources in a type-safe, reusable manner.

S3

Private bucket for static website hosting. No public access - secured with Origin Access Control.

CloudFront

Global CDN with HTTPS, HTTP/2, and HTTP/3 support. Provides low-latency access worldwide.

Route 53

DNS management with automatic record creation for CloudFront distributions.

ACM

SSL/TLS certificates with automatic DNS validation and renewal.

CodePipeline

Fully managed CI/CD service that automates build and deployment on every git push.

CI/CD Workflow

When I push code to GitHub, the following happens automatically:

1

Source

CodePipeline detects the push via GitHub CodeConnections and pulls the latest code.

2

Synth

CDK compiles TypeScript infrastructure code into CloudFormation templates.

3

Beta Deployment

Deploys to beta.resume.tako.tw for preview and testing.

4

Production Deployment

Automatically deploys to resume.tako.tw after beta succeeds.

Infrastructure Code

The entire infrastructure is defined in two CDK stacks:

WebsiteStack

Creates the hosting infrastructure for each environment:

const bucket = new s3.Bucket(this, 'WebsiteBucket', {
  blockPublicAccess: s3.BlockPublicAccess.BLOCK_ALL,
  encryption: s3.BucketEncryption.S3_MANAGED,
});

const distribution = new cloudfront.Distribution(this, 'CDN', {
  defaultBehavior: {
    origin: S3BucketOrigin.withOriginAccessControl(bucket),
    viewerProtocolPolicy: ViewerProtocolPolicy.REDIRECT_TO_HTTPS,
  },
  domainNames: [domainName],
  certificate: certificate,
});

PipelineStack

Defines the self-mutating CI/CD pipeline:

const pipeline = new CodePipeline(this, 'Pipeline', {
  synth: new ShellStep('Synth', {
    input: CodePipelineSource.connection(repo, branch, { connectionArn }),
    commands: [
      'cd infra && npm ci',
      'npm run build',
      'npx cdk synth',
    ],
  }),
});

pipeline.addStage(new WebsiteStage(this, 'Beta', { domainPrefix: 'beta' }));
pipeline.addStage(new WebsiteStage(this, 'Prod'));

Security Features

  • Private S3 Bucket - No public access, content served only through CloudFront
  • Origin Access Control (OAC) - Latest AWS-recommended method for S3 + CloudFront
  • HTTPS Enforced - All HTTP requests redirect to HTTPS
  • S3 Managed Encryption - Data encrypted at rest

Cost

~$1-2/month for this entire setup with low traffic. S3 and CloudFront have generous free tiers.
